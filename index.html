Before playing the game, please make sure to follow the installation instructions below (for macOS configuration):

  1. Make sure Homebrew is installed on your machine. If not, you can install it from the official Homebrew website.
  2. Install dependencies by running the following commands in the terminal:
    • brew install sdl2 sdl2_ttf
    • brew install python3
  3. Install Pybind11 using pip:
    • pip install pybind11
  4. Check your Python version and run the following command to build the game engine:
    • python3.9 macbuild.py
  5. Add the SDL2 framework to the /Library/Frameworks folder:
    • Download the SDL2 framework from the official SDL website.
    • Extract the .zip file and copy the SDL2.framework folder to the /Library/Frameworks folder on your machine.
  6. Finally, run the game with the following command:
    • python3.9 test.py -m ./mygameengine.so
